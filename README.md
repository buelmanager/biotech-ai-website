# BioTech AI Website

생명과학과 AI의 융합을 시각적으로 표현한 하이테크 바이오 테크놀로지 컨셉의 웹사이트입니다.

## 주요 기능

- **Particle Network Animation**: Canvas API를 활용한 인터랙티브 배경 애니메이션
- **GSAP ScrollTrigger**: 스크롤 기반 애니메이션 효과
- **Lenis Smooth Scroll**: 부드러운 스크롤 경험
- **Glassmorphism UI**: 반투명 유리 효과의 현대적 UI
- **DNA Helix Animation**: SVG 기반 DNA 구조 시각화
- **Network Visualization**: 데이터 네트워크 연결 애니메이션
- **Counter Animation**: 숫자 카운팅 효과
- **Responsive Design**: 모바일 최적화

## 기술 스택

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- GSAP 3.12 + ScrollTrigger
- Lenis Smooth Scroll
- Canvas API

- **컬러 팔레트**: 딥 네이비 배경 + 사이언 블루/네온 그린 포인트
